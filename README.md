# Ipoteka Bank OTP Group Theme

A custom JetBrains IDE theme plugin for Ipoteka Bank OTP Group.

## Installation

1. Download the plugin from JetBrains Marketplace or build from source
2. Go to **Settings/Preferences** > **Plugins** > **Install Plugin from Disk**
3. Select the plugin JAR file and restart the IDE
4. Go to **Settings/Preferences** > **Appearance & Behavior** > **Appearance**
5. Select "Ipoteka Bank OTP Group" from the Theme dropdown

## Color Palette

The theme is based on the official Ipoteka Bank brand colors:

| Color Name | Hex Code | Preview | Usage |
|------------|----------|---------|-------|
| Dark Green | `#006548` | ![#006548](https://img.shields.io/badge/-%23006548-006548) | Primary brand color, main UI elements |
| Lime Green | `#76BC21` | ![#76BC21](https://img.shields.io/badge/-%2376BC21-76BC21) | Accent color, highlights |

### Color Details

- **Dark Green (#006548)**: The primary brand color representing stability and trust. Used for main structural elements in the icon.
- **Lime Green (#76BC21)**: A vibrant accent color symbolizing growth and innovation. Used for the central focus point in the icon.

## Building and Deploying

For detailed instructions on building and deploying theme plugins, see the official documentation:
[Deploying a Theme](https://plugins.jetbrains.com/docs/intellij/deploying-theme.html)

### Quick Start

1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Go to **Build** > **Prepare Plugin Module for Deployment**
4. The plugin JAR will be generated in the project root directory

### Publishing to JetBrains Marketplace

1. Create an account on [JetBrains Marketplace](https://plugins.jetbrains.com/)
2. Generate a permanent token in your profile
3. Upload the plugin JAR file
4. Fill in the plugin details and submit for review

## Compatibility

- **Minimum IDE Version**: 2025.2 (Build 252.25557)
- **Supported IDEs**: IntelliJ IDEA, WebStorm, PyCharm, and other JetBrains IDEs
